# General Agentic Task Workflow

Execute the requested task using available tools. No multi-phase workflow required.

---

## Phase: execute | 执行 | Execution

Complete the task as specified by the user. Use tools as needed.

### Focus
Direct task execution. Plan → execute → verify → report.

### Gate Criteria
- [ ] REQUIRED: Task objective clearly understood before starting
- [ ] REQUIRED: Task completed or progress clearly communicated to user
